IfxAlchemy
==========

The maintained project documentation is in ``README.md``.

Compatible with SQLAlchemy 2.0.x for the public ``informix+pyodbc`` dialect.
SQLAlchemy 2.1 is validated experimentally until the suite is green and the
dependency range is widened. The package currently declares SQLAlchemy
``>=2.0,<2.1``. The legacy IfxPy module remains importable for source
compatibility but is not a published entry point.
