IfxAlchemy
==========

The maintained project documentation is in ``README.md``.

This package supports SQLAlchemy ``>=2.0,<2.1`` for the public
``informix+pyodbc`` dialect. SQLAlchemy 2.1 prereleases are tracked in
non-blocking CI until SQLAlchemy 2.1 is final and the suite is green. The
legacy IfxPy module remains importable for source compatibility but is not a
published entry point.
